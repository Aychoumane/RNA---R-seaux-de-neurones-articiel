Je vous présente mon projet qui est la création d'un réseau de neurones en C. 

Il inclut un réseau de neurone constitué d'une couche entrée, une couche caché, et d'une couche sortie. 
Afin d'améliorer les performances du réseau, il a été adapté afin qu'il supporte le multi-threading. 

Pour comprendre d'avantage les réflexions, ainsi que le code plus en profondeur, un fichier pdf nommé rna_explication.pdf 
est disponible dans le projet.
